==== 자바 API 도큐먼트

11장 부터는 자바에서 제공하는 API(Application Programming Interface)를 배운다.

API는 라이브러리라고 부르기도 하는데, 프로그램 개발에 자주 사용되는 클래스 및 인터페이스의 모음을 말한다.

String 클래스와 System 클래스도 모두 API에 속하는 클래스들이다.
