==== 인터페이스의 역할

자바에서 인터페이스는 객체의 사용 방법을 정의한 타입이라고 말할 수 있다.

인터페이스는 객체의 교환성을 높여주기 때문에 다형성을 구현하는 매우 중요한 역할을 한다.
